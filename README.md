# Log Analyzer

## Description

A Python-based Log Analyzer that processes log files and identifies important events such as errors, warnings, and informational messages.

## Features

* Log file parsing
* Error detection
* Event counting
* Information extraction
* Command-line interface

## Technologies Used

* Python 3
* File Handling
* Kali Linux

## How to Run

```bash
python3 log_analyzer.py
```

## Learning Outcomes

* Log analysis
* File processing
* Data extraction
* Security monitoring
